An **Attendee** is anyone who has attended an ETH Boulder event. This is the entry membrane — there is no other way to join the network.

## Entry Requirements

| Requirement | Verification |
|-------------|--------------|
| **Attend ETH Boulder event** | Check-in at event |
| **Verify identity** | Wallet signature or event credential |

## Rights

| Right | Description |
|-------|-------------|
| **Knowledge Graph contribution** | Create entities, add relationships, document learnings |
| **Community participation** | Join discussions, attend activities, contribute to projects |
| **Member nomination eligibility** | Can be nominated for membership after demonstrating participation |

## Responsibilities

| Responsibility | Description |
|----------------|-------------|
| **Respect the community** | Uphold network values in interactions |
| **Contribute authentically** | Add genuine value to the Knowledge Graph |
| **Participate in good faith** | Engage constructively with the network |

## Knowledge Graph Permissions

| Permission | Allowed |
|------------|---------|
| Read | ✅ All public entities |
| Write | ✅ Create entities, add relationships |
| Curate | ❌ Cannot reconcile or edit contested |
| Moderate | ❌ Cannot revert or protect |

## Progression

To become a **Member**:
1. Demonstrate ongoing participation after event
2. Receive nomination from existing Member
3. Pass 48h consent window with Member Assembly
